# Personal Portfolio Website

A modern, responsive, and production-ready portfolio website showcasing the profile, projects, skills, education, and credentials of **Pragadheeswaran S**.

This project is built using React.js, Vite, HTML5, CSS3, React Icons, and Framer Motion.

## 🚀 Live Demo & Visuals

- **Dark Mode by Default**: Tailored space-dark themed background with smooth glowing accents.
- **Glassmorphism Design**: Frosted glass panels (`backdrop-filter`) with thin borders and soft glow elevations.
- **Mobile-First Responsiveness**: Handheld, tablet, and desktop layout optimization.
- **Micro-Animations**: Scroll reveal cards, blinking cursor typists, slide-in timelines, and interactive hover feedback.

## ⚙️ Setup and Installation

Follow these steps to run the portfolio website locally.

### Prerequisites

Ensure you have **Node.js** (v18 or higher) and **npm** installed.

### Installation

1. Clone or download this project directory.
2. Navigate into the project folder:
   ```bash
   cd portfolio-website
   ```
3. Install package dependencies:
   ```bash
   npm install
   ```

### Execution

Start the development server:
```bash
npm run dev
```

The app will run locally. Click the link shown in the terminal (typically `http://localhost:5173`) to view the application in your browser.

### Build

Compile assets for production:
```bash
npm run build
```
The output files will be bundled inside the `dist/` directory.
